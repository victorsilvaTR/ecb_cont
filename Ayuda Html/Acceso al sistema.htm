Acceso al sistema

La ventana que apreciamos a continuación, aparecerá cada vez que se desee acceder al Sistema de Contabilidad del ECB 

 
  Ingreso al sistema    
 

Descripción

 

1. Usuario                   

Digite el nombre del usuario para el acceso al sistema.

2. Contraseña             

Digite la contraseña de su usuario Asignado.

3. Ingresar                   

Presione la tecla ENTER o hacer clic  en el botón para ingresar al sistema contable.

 

Posteriormente se despliega una ventana solicitando validar la Empresay el Año de Trabajo

 

 

Selección de empresa y año contable

Descripción

 

1. Empresa                 

Seleccione la empresa, para acceder al sistema contable.

2. Año                         

Seleccione el Año para acceder al Sistema Contable.

3. Ingresar                   

Presione la tecla ENTER ó hacer Clic en el botón para ingresar al Sistema Contable.
