MANTENIMIENTO DE USUARIO

 

Nos permite registrar usuarios y definirles su nivel de acceso al sistema.

  


 Listado de usuarios


 

Descripción

1. Filtro           

Ingrese el nombre del usuario a filtrar en la lista

2. Lista            

Muestra a los usuarios y sus privilegios


 

Al momento de crear un nuevo usuario o modificar los datos de un usuario existente se mostrara la siguiente interfase, permitiendo ingresar sus datos.

 


 Registro de usuarios



Descripción

1. Datos                      

Ingrese los datos del usuario como privilegio, nombre, cargo, clave

2. Activo                     

Active esta casilla para identificar el estado del usuario.

3. Administrador         

Active esta casilla para identificar si el usuario es de tipo administrador de sistema.
