Configuracion de saldos iniciales 

 

Permite configurar los saldos iniciales para el cálculo del flujo de efectivo, estos datos serán llenados siempre y cuando no se tenga información ingresada en el sistema del ejercicio a comparar.

 

 

 

Fig.  53 - Configuración  de los saldos iniciales

 

Descripción

 

1. Ejercicio     

Indica el ejercicio a configurar.

2. Periodo                   

Indica el periodo a configurar.

3. Cargar Datos           

Muestra la configuración almacenada.

4. Grabar                    

Almacena la configuración establecida.

5. Eliminar Todo        

Permite eliminar todos los datos de la lista mostrada

6. Salir                        

Permite salir del formulario actual

7. Lista                        

Muestra los datos por ejercicio y periodo seleccionado
