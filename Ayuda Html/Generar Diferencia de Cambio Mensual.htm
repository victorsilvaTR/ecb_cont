GENERAR DIFERENCIA CAMBIO MENSUAL

 

En este proceso nos permite generar la diferencia de cambio mensual de las cuentas de activo y pasivo, para ello debemos seleccionar el libro que se generara el asiento automático y el periodo; además debemos ingresar el tipo de cambio de compra y venta de cierre del mes seleccionado, estos tipos de cambio actualizará el maestro de tipos de cambios (Configuración à Tipo de cambio)

 



Configuración de diferencia de tipo de cambio mensual


Descripción

 

1.       Libro                           

Libro diferencia de Cambio Mensual

2.       Mes                             

Se selecciona el mes que se desea ajustar.

3.       Tipo de Cambio Compra  

Muestra el tipo de cambio de compra mensual por el mes seleccionado.

4.       Tipo de Cambio Venta

Muestra el tipo de cambio de venta mensual por el mes seleccionado.

5.       Generar                       

Inicia el proceso, generando el asiento por diferencia de cambio.

6.       Eliminar

Elimina los asientos automáticos de diferencia de cambio.

7.       Salir

Permite salir del modulo de diferencia de cambio.
