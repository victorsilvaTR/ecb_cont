Configuración del cuentas adicionales

Con este formulario podemos agregar cuentas mayores a dos dígitos, al proceso y al reporte de flujos de efectivo que esta definido a dos dígitos, esto se utiliza si se desea tener un mayor detalle del flujo de determinadas cuentas.

 

El formulario contiene un filtro por periodos y el campo cuenta que se utiliza como un filtro para la búsqueda de cuentas a adicionar en la lista.

 


Fig.  48 - Configuración de cuentas adicionales para el Flujo de Efectivo

 

Descripción

 

1. Periodo                   

Indica el periodo a configurar.

2. Cuenta                    

Ingrese el numero de cuenta utilizarlo como filtro en la búsqueda de cuentas.

3. Buscar

Permite buscar las cuentas según el filtro de cuenta ingresado.

4. Listar          

Muestra la configuración almacenada.

5. Grabar                    

Almacena la configuración establecida.

6. Eliminar item          

Permite eliminar la cuenta seleccionada de la lista

7. Ordenar

Permite ordenar la lista de cuentas

8. Salir                        

Permite salir del formulario actual

9. Lista            

Muestra los datos configurados por periodo seleccionado
