1. - Objetivo del Manual

El Manual  tiene como objetivo  dar  a conocer al usuario operativo el funcionamiento sostenido de las  herramientas,  para el manejo operativo del Sistema Contable,  para lograr un  óptimo  manejo y performance del trabajo a realizar.

El usuario encontrará en el Manual, el detalle y la descripción de las Ventanas de cada Módulo, además, soluciones  prácticas frente a problemas que se pudieran suscitar, alternativas de los   procedimientos a emplear para su respectiva subsanación.
