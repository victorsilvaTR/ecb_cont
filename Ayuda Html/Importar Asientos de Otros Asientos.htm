IMPORTAR DATOS DE PLANTILLA XLS

 


Importar Asientos de otros Asientos

 

 

Descripción

 

1.       Selección de archivo

Permite abrir un cuadro de dialogo para buscar archivo que se desea abrir y en la caja de texto muestra la ruta y el nombre del archivo

2.       Ruta

Muestra la ruta y el nombre del archivo

3.       Opciones

Seleccione los datos que se importaran

4.       Progreso

Muestra el progreso de la importación

5.       Importar datos

Permite dar el inicio de la importación de los datos

6.       Salir

Permite salir del formulario
