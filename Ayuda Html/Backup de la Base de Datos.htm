BACKUP DELA BASE DE DATOS

Nos permite realizar un backup de la base de datos, la ruta o directorio donde se almacenara el backup se determina en el archivo de configuración de sistema.

 


Backup de la base de datos

 

Descripción

1. Backup       

Inicia el proceso de backup de toda la base de datos

2. Salir                        

Permite salir del formulario de Backup

 

NOTA: De preferencia la ruta del backup es de una carpeta compartida para que puedan acceder los usuarios del sistema. La ruta del directorio compartido debe asignarse en el archivo de configuración inicial del sistema.
