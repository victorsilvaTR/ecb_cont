Exportar datos DAOT

Nos permite exportar los datos del DAOT a un archivo de tipo DBF con la estructura especificada porla SUNAT

 


 Exportar datos DAOT


Descripción

1. Unidad                    

Seleccione la unidad de disco en donde se almacenara los datos exportados

2. Directorios              

Muestra las carpetas o directorios donde se guardaran los datos a exportar.

3. Ruta                        

Muestra la ruta completa de la carpeta seleccionada

4. Datos del DAOT      

Ingrese los datos para la exportación del DAOT, como el tipo de entidad, el monto mínimo, los rangos de fecha y el tipo de moneda

5. Exportar Datos        

Inicia el proceso de exportación de las opciones seleccionadas
