REGISTRO DE INTANGIBLES

 

 

Registro de intangibles

 

Descripción

 

1.   Período                 

Seleccione el mes donde se buscaran los Vouchers.

2.   Listar                    

Esta opción permite refrescar una modificación y recuperar la información previamente guardada.

3.   Grabar                  

Esta opción permite  grabar el registro de las operaciones

4.   Insertar Mov          

Esta opción permite insertar un movimiento de intangibles

5.   Eliminar Item        

Esta opción permite eliminar un registro seleccionado.

6.  Eliminar Todo        

Esta opción permite eliminar en forma masiva todos los registros.

7.  Salir                       

Esta opción permite salir del formulario

8.  Listado                   

Muestra movimiento de Vouchers
