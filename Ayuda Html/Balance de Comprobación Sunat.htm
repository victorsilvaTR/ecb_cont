Balance de Comprobación 



Reporte de balance de comprobación


Descripción

1. Balance Acumulado           

Seleccione esta opción para mostrar un reporte acumulado al periodo seleccionado

2. Balance del mes                 

Seleccione esta opción para mostrar un reporte del periodo seleccionado

3. Dígitos                                

Indica el numero de dígitos de las cuentas a mostrar en el reporte

4. Mes                                    

Seleccione el periodo a filtrar en el reporte

5. Moneda                              

Indica el tipo de moneda del reporte

6. Imprimir                             

Muestra el reporte en pantalla según la configuración elegida

7. Salir                                   

Permite salir del reporte.
