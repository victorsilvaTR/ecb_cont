Configuracíón del reporte 

 

Permite configurar el proceso del flujo de efectivo, en la figura adjunta se muestra la descripción de las actividades, en este formulario se seleccionara las columnas y las cuentas de donde se obtendrán los valores calculados, si la cuenta esta en blanco entonces el valor calculado será la suma de la columna seleccionada.

 

Se deberá configurar la formulas para cada ítem del reporte. En la columna de tipo de formula, existen dos tipos de ajuste Manual, Hoja de trabajo.

 

a) Cuando se selecciona el Tipo de Ajuste Manual, el usuario podrá ingresar un monto específico que será calculado en el proceso y emitido en el reporte.

 

b) Cuando se selecciona el Tipo de Ajuste de Hoja de Trabajo se deberá configurar una formula, haciendo clic en el símbolo (+) del lado derecho, e inmediatamente se visualizará un formulario de diseñador de formulas (La descripción del formulario se encuentra en el punto anterior).

 

 

Fig.  52 - Configuración del Reporte de Flujo de Efectivo

 

Descripción

1. Periodo                   

Indica el periodo a configurar.

2. Método                   

Seleccione el tipo de método de flujo de efectivo a configurar.

3. Opción de importar 

Permite visualizar la opción de importación.

4. Periodo       

Seleccione el periodo de origen para la importación, el periodo de destino es el periodo de configuración del proceso

5. Importar      

Inicia el proceso de importación de los periodos seleccionados.

6. Listar                      

Muestra los datos almacenados en el periodo seleccionado.

7. Grabar                    

Almacena la configuración establecida.

8. Eliminar datos

Permite eliminar los datos de la cuenta seleccionada

9. Eliminar Todo        

Permite eliminar todos los datos de la lista mostrada

10. Salir                                  

Permite salir del formulario actual

11. Lista                      

Muestra los datos configurados por tipo de método y periodo seleccionado
