Menús Principales

El sistema consta con las siguientes secciones:

 

Configuración                    

Aquí se determina la configuración del sistema.

 

Ingresos                              

En esta parte se ingresan los asientos contables, los presupuestos, las importaciones y exportaciones del

COA

 

Registros Auxiliares           

En esta sección se ingresan los registros auxiliares de compras y ventas

 

Procesos                             

En esta parte se realizan los procesos de diferencia en cambio, el cierre del ejercicio, el asiento de apertura, el

asiento de cierre final, el cierre mensual y el proceso de conversión de moneda.

 

Conciliación Bancaria      

Aquí se registra el extracto bancario y se emiten los reportes respectivos, como movimientos de bancos,

reportes de cheques emitidos y reporte de cheques pendientes.

 

Reportes de Análisis                                

En esta sección se muestran todos los reportes del sistema, estos serán detallados mas  adelante.

 

Reportes SUNAT                           

En esta sección se muestran todos los reportes que la SUNAT solicita, según los formatos requeridos.

 

Gerencial                            

Permite registrar los ratios financieros y mostrar el reporte respectivo según la configuración del usuario.

 

Utilitarios

En esta parte del sistema tienen opciones de exportación e importación de data como también la generación y

restauración del backup de toda la base de datos. También se puede configurar los accesos al sistema y la

creación de empresas.

 

Ayuda                            

En esta sección se muestra las diferentes formas de ayuda para que el usuario utilice el sistema contable.
