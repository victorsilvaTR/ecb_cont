LIBROS

 

En esta opción se determina los libros con los que se desea trabajar por ejemplo: libro de compras, ventas, diario, caja y otros.

 

Ventana configuración de libros

 

Descripción

1.  Filtro           Ingrese el nombre del libro a filtrar

2.  Lista            Muestra la lista de libros creados en el sistema

 

Mantenimiento de libros

 

Descripción

1.   Código                 

Se Registra el Código del Libro. 

2.   Código Sunat       

Se Registra el Código del Libro según SUNAT.

3   Nombre Largo       

Se consigna la descripción del Libro.
