TIPOS DE DOCUMENTO 

 

En esta Opción se determina los tipos de documentos del sistema.

 

 

 

Fig. 18. Listado de documentos

 

Descripción Fig. 18:

 

1. Filtro                        Ingrese el nombre del documento a filtrar en la lista

2. Lista                         Muestra la lista de documentos creados en el sistema indicando si serán incluidos en el DAOT y el tipo de operación si es negativo o positivo

 

 

 

 

Fig.19. Mantenimiento de tipos de documento

 

Descripción Fig. 19:

 

1.   Código                  El Código se asigna según el Tipo de Documento.

2.   Nombre Largo       Nombre del Documento.

3.   Nombre Corto        Nombre corto del Documento.

4.   Incluir en DAOT     Opción para incluir en el DAOT.

5.  Operación DAOT    Indicador para el DAOT.
