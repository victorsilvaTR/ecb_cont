TIPOS DE ASIENTO 

 

Esta opción nos permite configurar y personalizar los asientos Tipo, segúnla Naturaleza de las operaciones, que son de uso permanente, facilitando al usuario con el Ingreso de Datos del Documento, generar Automáticamente el Asiento Contable.

 

 

 

 

 

Fig. 40. Listado de los tipos de asiento

 

 

 

 

Fig. 41. Mantenimiento de los tipos de asiento

 

 

Descripción Fig. 41:

 

1.   Diario                    Se selecciona el Libro donde permita el Registro.

2.   Código                   El Sistema genera Automáticamente según la posición del Libro.

3.  Descripción Se consigna la descripción del Asiento Contable.

4.  Cuenta                    Se selecciona la Cuenta Contable.

5.  Tipo                                    Se selecciona si la Cuenta es Debe ó Haber.

6.   Porcentaje              Esta Opción nos permite distribuir en Porcentajes la Cuenta Deudora ó Acreedora.

7.   Insertar Item           Esta Opción nos permite Registrar la Cuenta con los datos consignados.

8.  Eliminar Item          Esta Opción nos permite Eliminar la Cuenta Registrada y poder registrar otra.
