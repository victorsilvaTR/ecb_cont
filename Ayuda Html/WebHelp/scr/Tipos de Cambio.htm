TIPO DE CAMBIO 

 

En este formulario se registran los distintos tipos de Cambio (Compra Vigente, Venta Vigente, Venta Publicación) dela Moneda extranjera y moneda adicional por día según sea necesario.

También se ingresaran los tipos de cambio mensual como el tipo de cambio de compra cierre, venta cierre y promedio Mensual.

A su vez el Sistema contempla el Ingreso de Tipos de Cambio Permanente siempre que se defina como Tipo de Contabilidad Bimoneda, es decir, sí el Tipo de Contabilidad es nacional sólo solicitará Tipo de Cambio para las operaciones en Moneda Extranjera para su respectiva Conversión.

Descripción Fig. 34:

1.   Moneda                             Se selecciona la Moneda.

2.   Fecha                                Se Registra la fecha del Tipo de Cambio

3.  Compra Vigente                  Se Registra el Tipo de Cambio Compra Vigente.

4.  Venta Vigente                     Se Registra el Tipo de Cambio Venta Vigente.

5.   Venta Publicación             Se Registra el Tipo de Cambio Venta Publicación.

 

En esta Opción se ingresa los Tipos de Cambio Promedio para las  Conversiones Automáticas de una Contabilidad en Moneda Nacional a Bimoneda.

 

 

Fig. 35. Mantenimiento de tipos de cambio mensuales

 

Descripción Fig. 35:

1.   Tipo                       Seleccione el tipo de cambio mensual a grabar (Compra cierre, Venta cierre y Promedio mensual)

2.   Moneda                 Seleccione el tipo de moneda

3.   Mes                        Ingrese los tipos de cambio por mes

4.   Grabar                   Grabar los tipos de cambio ingresados.
