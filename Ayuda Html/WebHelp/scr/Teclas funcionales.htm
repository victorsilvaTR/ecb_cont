Teclas Funcionales

A continuación se detallan las teclas funcionales para los tipos de ventanas que se encuentran en el sistema.

Las teclas de función del F1 al F7 están habilitadas para cada ventana que tenga la barra de herramientas.

 

F1        Permite Desplegar una Ventana de Ayuda.

F2        Permite Ingresar  un Nuevo Registro.

F3        Permite Consultar los Datos de un Registro.

F4        Permite Grabarla Información de un Nuevo Registro o Modificación del Mismo.

F5        Permite Borrar un Determinado Registro.

F6        Permite Modificar los Datos de un Registro.

F7        Permite visualizar el reporte de la ventana seleccionada

 

 

Para el formulario de ingreso de asientos están habilitadas todas las teclas de función del F1 al F12, adicionalmente la tecla de escape.

 

F8        Permite Desplegar una Ventana de Asiento Tipo.

F9        Permite Asignar el IGV Automático.

F10      Permite Copiar el Importe del Acumulado de Debe o Haber.

F11      Permite Repetir los Conceptos del Documento del Registro Anterior.      

F12      Permite Repetir un Campo del  Documento del Registro Anterior.

ESC     Permite Salir del Sistema.
