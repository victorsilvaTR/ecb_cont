REGISTRO DE ASIENTOS CONTABLES

 

En este módulo se registran todas las operaciones de la empresa.

 

 

Fig. 53. Listado de Registro de Asientos

 

Descripción Fig. 53:

1. F iltrar Datos                        Sirve para efectuar una rápida búsqueda de un registro contable. Ya sea esta por. Libro, Voucher ó Glosa, etc.

2.  Listar                                    Sirve para refrescar el módulo de registros contables.

3.  Período                               Sirve para indicar el mes del registro contable, modificación, eliminación ó consulta del mismo.

 

El formulario de registro de asientos contables tiene las siguientes características.

 

 

Fig. 54. Formulario de registro de asientos contables

 

Descripción Fig. 54:

1.   Grabar                   Esta opción permite grabar el registro contable.

2.   Cancelar                Esta opción permite cancelar un nuevo registro contable, modificación ó consulta del mismo.

3.   Estado                   Indicador de estado actual del registro contable, si es de consulta, nuevo o edición de datos.

4.   Libro                      En esta opción se selecciona el libro donde se desea registrar la operación.

5.  Voucher                  Este es un control numérico del voucher y esta compuesto por: código de libro, mes y el número que le corresponde al registro; el sistema asigna automáticamente un número correlativo.

6.   Fecha                     Se registra la fecha contable.

7.   Moneda                  Se selecciona esta opción según la naturaleza de los documentos.

8.   Glosa                     Se registra la glosa principal de todo el asiento contable.

9.   IGV (%)                E n esta opción muestra el porcentaje del IGV, permitiendo modificarlos si es necesario.

10. Cuenta                   Se registra la cuenta contable que este asociado al libro, a su vez se puede hacer una búsqueda rápida con F1 y registrar la misma.

11. Código de Entidad Esta opción nos permite controlar por cuentas corrientes, Cuentas  por Cobrar ó Pagar, siempre que este previamente configurado la cuenta en el plan contable y tenga la opción de entidad activada.

 12. Centro de Costos   Esta opción nos permite controlar el gasto ó costo, siempre que este configurado la cuenta en el plan contable con centro de costo. Los gastos sólo se configuran por la clase 6 ó 9 dependiendo como se va procesar la información contable.

13. Comprobante         En esta opción se registran los datos del comprobante como son: Tipo de Documento, Serie y Número.

14. Fecha del Dcmto    Se registra la fecha de emisión del comprobante.

15. Fecha Vcmto         Se registra por fecha de vencimiento ó pago, cuando los comprobantes son generalmente de servicios.

16. Moneda                  Se registra el símbolo de la moneda; el sistema asume  por defecto inicial la opción (7).

17. Operación. TC        En esta opción se asignan los tipos de operación como Sin conversión SCV, Compra vigente COM, venta vigente VEN, Venta Publicación VEP, según la naturaleza del tipo de moneda de todo el voucher.

18. Tipo de Cambio     En esta opción el sistema valida los tipos de cambio registrados en la tabla de Configuración - Tipo de Cambio.

19. Debe y Haber Nac.  Se registra el importe de las operaciones de la moneda nacional

20. Debe y Haber Ext.   Se registra el importe de las operaciones del amoneda extranjera.

21. Barra de Estado      Muestra al usuario la descripción de la  cuenta, entidad, centro de costos y tipo de documento seleccionado por el usuario.

22. Otras Monedas       Esta opción se utiliza para registrar otras monedas como: euros, francos, libras, etc. Se debe indicar el tipo de moneda.

23. Importe                  Se registra el importe de otras monedas, el cual será validado con la tabla de tipos de cambio – Monedas Adicionales.

24. Glosa                     El sistema repite la glosa principal ingresado y que puede ser cambiado por el usuario según el concepto de la cuenta.

25. Calculo Manual      Se activa esta opción para el cálculo manual, de todo el voucher según sea necesario.

26. Inafecto                  Se activa esta opción para las operaciones inafectas.

27. Base Imponible      En esta opción se registra el tipo operación como son las Operaciones Gravadas (a), Gravada y No Gravada(b) y Sin derecho a Crédito Fiscal(c).

28. Pago de IGV           Esta opción se activa para las operaciones afectas a detracciones, percepciones o retenciones según la configuración de la cuenta en el plan de cuentas.

29. Dcmto. Referencia  En esta opción se registran las facturas asociadas a notas de crédito y debito. Como son Tipo de documento, serie, Número documento y fecha del documento.

30. Transacción           Esta opción se utiliza para describir el tipo de transacción de las retenciones (como agente retenedor) y percepciones (como agente perceptor).

31. Constancia             Una vez activado la opción 28, se activa esta opción permitiendo ingresar el número de constancia y la fecha de la operación.

32. Diferencia               En esta opción nos muestra la diferencia del asiento contable del debe y haber de la moneda nacional y extranjera, en caso de que exista una diferencia  el sistema no permite grabar el voucher como consistencia. El sistema contable tiene dos formas de validar, si  la contabilidad esta predeterminado en moneda nacional (soles)  el sistema sólo validará  las columnas del debe y haber de soles, y si estuviera en el tipo de contabilidad bimoneda, el sistema validará debe y haber de soles y dólares.

33. Eliminar                 Esta opción nos permite eliminar un registro de un asiento contable.

34. Provisión                Esta opción sirve para migrar los comprobantes provisionados al formulario de registro de asientos contables para realizar un pago, amortización, canje o compensación del mismo.

35. Asientos                 En esta opción nos permite utilizar los asientos tipos previamente registrados  en el módulo Configuración – Asientos Tipos.

36. Verificar                  Esta opción nos verifica la diferencia entre el debe y haber de la moneda nacional; cuando el tipo de moneda del voucher esta en dólares. Si se encuentra una diferencia se agrega automáticamente la cuenta y el importe por diferencia de cambio.

37. Entidad                   Esta opción nos permite el registro en línea de una nueva Entidad, estando en el asiento contable,  permitirá ejecutar dicha opción siempre que este configurado con la opción entidad  en el plan de cuentas.

38. Barra de ayuda       Esta es una barra de ayuda y muestra las siguientes opciones: F8 Tipo de Asiento, F9 IGV Automático, F10 Copia Acumulada, F11 Repetir Todo, F12 Repetir Anterior
