Registros Auxiliares 

 

Este módulo nos permite registrar documento por documento las operaciones con boleta de venta y mantener un enlace con el número de registro contable y el importe que fueron registrados en forma grupal en el registro de compras y ventas.

 

 

Fig. 67. Ingreso de registros auxiliares de Compras y Ventas

 

 

Fig. 68. Registro de una Boleta de Venta

 

Descripción Fig. 68:

1.       Barra de Herramientas     En la barra herramientas tenemos las siguientes opciones: NUEVO, sirve para registrar un nuevo registro auxiliar. ELIMINAR, sirve para eliminar un registro previa selección horizontal del mismo. IMPRIMIR, sirve para imprimir un registro auxiliar. SALIR, sirve para salir del módulo del registro auxiliar.

2.       Voucher                          Nos muestra el número de registro principal de compras y ventas.

3.       Moneda                           Nos muestra el tipo de moneda del registro principal, compras ó ventas.

4.       Total                               Nos indica el importe global del registro principal, compras ó ventas.

5.       Documento                     Nos indica el número de voucher del registro principal, compras ó         ventas.

6.       Fecha                              Nos indica la fecha del voucher del registro principal, compras ó ventas.

7.       Saldo                              Este es un controlador de saldo con los importes registrados previamente.

8.       Tipo de documento         Esta predeterminado para mostrar Boleta de Venta.

9.       Serie                               Ingrese el número de serie.

10.   Número                           Ingrese el número de documento.

11.   Moneda                           Se selecciona el tipo de moneda.

12.   Fecha Documento           Se registra la fecha del documento.

13.   Tipo de Cambio              Se indica el tipo de cambio, siempre que sea dólares.

14.   Base Inafecta                   Se registra el importe inafecto de la boleta.

15.   Otros                               Se registra el importe de otros pagos, como: Servicios, comisiones, etc.

16.   Total                               Se registra el importe neto afecto.

17.   Tipo de Documento         Se registra el tipo documento.

18.   Número de Documento   Se registra el número de documento.

19.   Nombres                         Se registra el nombre y apellido de la persona.

20.   Dirección                        Se registra la dirección de la persona.

21.   Grabar                             Permite grabar la boleta ingresada.

22.   Cancelar                          Cancela el ingreso realizado.

 

Los importes registrados superiores al 50% UIT; el sistema solicitara registrar los datos mínimos de las casillas 17,18 y19 para permitir grabar, caso contrario, no se efectuara la grabación hasta ingresar los datos.
