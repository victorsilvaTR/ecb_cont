Entidades

 

Aquí se ingresan detalladamente los Clientes, Proveedores, Trabajadores indicando el tipo de entidad al cual pertenece.

 

 

 

Fig. 16. Listado de entidades

 

Descripción Fig. 16:

 

1.   Filtro          Ingrese el código, nombre o tipo de entidad a filtrar.

2.   Lista           Muestra las entidades creadas en el sistema.

 

 

 

 

Fig. 17. Mantenimiento de entidades

 

Descripción Fig. 17:

 

1.   Tipo de Entidad     Se selecciona el Tipo de Entidad, por ejemplo: Cliente, Proveedor, Trabajador.

2.   Código de Entidad El Sistema asigna Automáticamente un Código interno de Correlatividad.

3.   Descripción           Se Registra el Nombre ó Razón Social.

4.   Dirección               Se Registra la Dirección.

5.  Tipo de persona      Se selecciona el Tipo de persona Natural,  jurídica ó No Domiciliado.

6.  Tipo de Dcto.          Se asocia al Tipo de Documento RUC, DNI y Otros.

7.  Número de Dcto.     Se Registra el número de RUC, DNI y Otros.

8.  Representante         Se Registra el nombre del Representante Legal.
