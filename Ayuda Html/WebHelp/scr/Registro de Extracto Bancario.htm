REGISTRO DEL EXTRACTO BANCARIO

 

Este tiene dos procesos: Primero nos muestra el movimiento sin conciliar de un mes, el banco y la cuenta corriente del mismo permitiendo importarlos a los movimientos conciliados. Segundo una vez transferido los movimientos de una cuenta se realizan el proceso de conciliación bancaria, se graba y queda registrado en la ventana de movimiento conciliado.

 

 

Fig. 79. Movimiento sin Conciliar  de un  Banco – Cuenta Corriente

 

Descripción Fig. 79:

1.       En esta opción se selecciona el banco, para los efectos deberá estar registrado en el mantenimiento de Configuración de Bancos.

2.       En esta opción se selecciona la cuenta corriente creada anteriormente, en mantenimiento de Configuración de Cuenta Corriente.

3.       Se selecciona el periodo donde se realizará la conciliación bancaria.

4.       Esta opción refresca los registros consignados.

5.       Esta opción permite importar para el proceso de conciliación bancaria registro por registro.

6.       Esta opción permite importar para el proceso de conciliación bancaria todos los movimientos sin conciliar.

 

 

Fig. 80. Movimiento conciliado  de un  banco por cuenta corriente

 

Descripción Fig. 80

1.       En esta opción se selecciona el banco.

2.       En esta opción se selecciona la cuenta corriente.

3.       Se selecciona el mes para realizar la conciliación bancaria.

4.       Esta opción nos permite presionando F11 repetir la fecha del documento.

5.       Se registra las fechas de las operaciones consignadas en el extracto bancario según correspondencia de cada una de ellas.

6.       Se registra las observaciones de la operación, siempre que lo amerite.

7.       Mediante este proceso se  refresca la lista.

8.       Esta opción verifica si el voucher fue eliminado ó modificado en los importes en el registro contable.

9.       Esta opción nos permite eliminar un registro previa selección del mismo.

10.   Esta opción nos permite eliminar todos los  movimientos conciliados.

11.   Esta opción nos permite grabar la conciliación bancaria.
