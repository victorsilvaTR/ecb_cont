Objetivo

 

El Manual  tiene como objetivo  dar  a conocer al usuario el funcionamiento de las  herramientas del Sistema Contable,  para lograr un  óptimo  manejo y performance del trabajo a realizar.

El usuario encontrará en el Manual, el detalle y la descripción de las ventanas de cada módulo, además de soluciones  prácticas frente a problemas que se pudieran encontrar.
