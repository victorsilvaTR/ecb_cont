TIPO DE MONEDA 

 

En este formulario se define  los  tipos de moneda, que contará  el sistema contable, adicionalmente se indicará si la moneda creada será la moneda nacional o extranjera del sistema. También se configura si la contabilidad a llevar será una contabilidad en moneda nacional o una contabilidad bimoneda.  

 

 

Fig. 31. Listado de tipos de moneda y configuración de tipo de contabilidad

 

Descripción Fig. 31:

 

1.   Contabilidad en Moneda Nacional             Marcando esta Opción se define el Tipo de Contabilidad en Moneda Nacional por ejemplo: (Soles)

2.   Contabilidad en Moneda Bimoneda           Marcando esta Opción se define el Tipo de Contabilidad en Bimoneda por ejemplo: (Soles y Dólares)

3.   Descripción                                               Ingrese la descripción a filtrar en la lista

4.   Lista                                                          Muestra los tipos de moneda creados en el sistema

 

 

 

Fig. 32. Mantenimiento del tipo de moneda

 

Descripción Fig. 32:

 

1.   Código                  El Código se Asigna según el criterio del usuario.

2.   Descripción           Nombre de la moneda.

3.  Abreviatura             Consignar el símbolo.

4.  Monedas                 Seleccione si la moneda a crear será la moneda nacional o extranjera del sistema. Esta opción estará desactivada si ya fue asignada una moneda de este tipo.
