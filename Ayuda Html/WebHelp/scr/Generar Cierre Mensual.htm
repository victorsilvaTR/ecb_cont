GENERAR CIERRE MENSUAL

 

En este formulario se realiza el cierre del período, la ejecución del mismo no permitirá ingresar y modificar ningún voucher, solo se podrán realizar dichas operaciones si esta opción es modificada por el administrador autorizado a la opción Abrir el mes cerrado.

 

 

 

Fig. 76. Registro de Cierre Mensual

 

 

Descripción Fig. 76

 

1.       Cerrar Mes             :En esta opción nos permite ejecutar el cierre del mes

2.       Abrir mes cerrado : En esta opción nos permite abrir el mes cerrado.

3.       Libro                     : Seleccione el periodo que se abrirá o cerrará.

4.       Procesar               : En esta opción se ejecuta los procesos indicados.
