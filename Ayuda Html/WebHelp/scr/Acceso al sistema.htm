Acceso al sistema

La ventana que apreciamos a continuación, aparecerá cada vez que se desee acceder al Sistema de Contabilidad del ECB (Fig.01)

 

Descripción Fig. 01:

1. Usuario                    Digite el nombre del usuario para el acceso al sistema.

2. Contraseña              Digite la contraseña de su usuario Asignado.

3. Ingresar                    Presione la tecla ENTER o hacer clic  en el botón para ingresar al sistema contable.

 

Posteriormente se despliega una ventana solicitando validar la Empresa y el Año de Trabajo (Fig.02)

 

 

Descripción Fig. 02:

1. Empresa                  Seleccione la empresa, para acceder al sistema contable.

2. Año                          Seleccione el Año para acceder al Sistema Contable.

3. Ingresar                    Presione la tecla ENTER ó hacer Clic en el botón para ingresar al Sistema Contable.

NOTA: En la parte inferior de la ventana se muestra el nombre del usuario activo, el periodo y el año seleccionado, la versión del sistema y el nombre de la base de datos donde esta conectado el sistema. En la parte superior el nombre de la empresa (Fig. 03).
