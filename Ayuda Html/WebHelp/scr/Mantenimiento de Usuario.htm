MANTENIMIENTO DE USUARIO

 

Nos permite registrar usuarios y definirles su nivel de acceso al sistema.

  

 

 

Fig. 123. Listado de usuarios

 

Descripción Fig.123

 

1. Filtro            Ingrese el nombre del usuario a filtrar en la lista

2. Lista             Muestra a los usuarios y sus privilegios

 

Al momento de crear un nuevo usuario o modificar los datos de un usuario existente se mostrara la siguiente interfase (Fig. 124), permitiendo ingresar sus datos.

 

 

Fig. 124. Registro de usuarios

 

Descripción Fig.124

 

1. Datos           Ingrese los datos del usuario como privilegio, nombre, cargo, clave

2. Activo          Active esta casilla para identificar el estado del usuario.
